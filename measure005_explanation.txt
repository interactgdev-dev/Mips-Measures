Measure 005 (HF: ACE/ARB/ARNI for LVSD) - Implementation Notes

Source reference:
- 2026_Measure_005_MIPSCQM.pdf

What this code now does (denominator-focused):
1) Age check:
   - Includes only patients age >= 18.

2) Heart failure diagnosis check (ICD):
   - Uses HF ICD list from measure 005 denominator.
   - At least one matching ICD required.

3) Submission criteria handling:
   - Criteria 1 path (outpatient):
     - Requires outpatient encounter code from denominator list.
     - Requires LVEF <= 40 marker: M1150.
     - Excludes if transplant/LVAD marker present: M1151.
   - Criteria 2 path (discharge):
     - Requires discharge encounter code: 99238 or 99239.
     - Requires NO telehealth discharge marker: M1426.
     - Requires LVEF <= 40 marker: M1150.
     - Excludes if transplant/LVAD marker present: M1151.

4) Denominator exclusions:
   - Denominator exclusion applied: M1151
   - If M1151 is present, record is excluded from denominator.

5) Denominator eligibility:
   - Eligible if criteria 1 OR criteria 2 is satisfied.
   - Output mapping:
     - ICD5C1 / ICD5C2 = HF diagnosis flags
     - CPT5C1 = outpatient encounter flag
     - CPT5C2 = discharge encounter flag
     - M005 = 1 if denominator eligible, else 0
     - E005 = 0 if eligible, else 1

Numerator handling:
- Numerator logic intentionally removed for this measure.
- These fields are intentionally neutralized:
  - N005_MET = 0
  - N005_EXCEPTION = 0
  - N005_NOT_MET = 0
  - QDC005 = ""

Important note vs PDF:
- PDF requires an additional denominator encounter for criteria 1.
- Current implementation checks main encounter path but does not independently enforce "at least one additional encounter" count logic.
