Measure 006 (CAD: Antiplatelet Therapy) - Implementation Notes

Source reference:
- 2026_Measure_006_MIPSCQM.pdf

What this code now does (denominator-focused):
1) Age check:
   - Includes only patients age >= 18.

2) CAD diagnosis check (ICD):
   - Uses CAD ICD-10 denominator list from measure 006.
   - At least one matching ICD required.

3) Encounter check (CPT):
   - Uses denominator encounter CPT list from measure 006.
   - At least one matching encounter code required.

4) Denominator exclusions:
   - No explicit denominator exclusion codes are currently configured for this measure.
   - Structure exists in code to add exclusions later if needed.

5) Denominator eligibility:
   - Denominator met when BOTH are true:
     a) CAD ICD matched
     b) Encounter CPT matched
     c) No denominator exclusion code matched
   - Output mapping:
     - ICD6 = 1 if diagnosis matched, else 0
     - CPT6 = 1 if encounter matched, else 0
     - M006 = 1 if denominator eligible, else 0
     - E006 = 0 if eligible, else 1

Numerator handling:
- Numerator logic intentionally removed for this measure.
- These fields are intentionally neutralized:
  - N006_MET = 0
  - N006_EXCEPTION = 0
  - N006_NOT_MET = 0
  - QDC006 = ""

PDF alignment summary:
- Requested scope (age + denominator ICD/CPT only) is implemented.
- Numerator (aspirin/clopidogrel performance/QDC) is not being evaluated in this updated code.
